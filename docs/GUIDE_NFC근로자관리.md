# NFC 근로자 관리 사용 가이드
> 관리자(현장소장·공무) 대상 / 최초 담당자도 이 문서 한 장으로 운영 가능

---

## 개요

NFC 근로자 관리 시스템은 외국인 근로자에게 **NFC 스티커 한 장**을 발급하고,  
이후 모든 TBM 참석·안전교육 이수를 스마트폰 태그 한 번으로 처리하는 시스템입니다.

```
근로자 등록 → NFC 스티커 발급 → NFC 태그 1회 = 참석 자동 인증
```

**관리자 접속 경로:** `SAFE-LINK 앱 → 관리자 메뉴 → NFC 근로자 관리`

---

## 사전 준비

| 항목 | 내용 |
|------|------|
| 계정 권한 | 관리자(admin) 이상 |
| NFC 스티커 | NTAG213 규격 (시중 1,000원~2,000원) |
| NFC 쓰기용 기기 | Android 스마트폰 (NFC 지원 모델) |
| 근로자 정보 | 이름, 국적, 공종, 동의서 서명 여부 |

> iOS는 NFC 읽기는 가능하나 **쓰기는 Android만 가능**합니다.

---

## Step 1. 근로자 신규 등록

**경로:** `관리자 → 근로자 등록` (`/admin/workers/enroll`)

### 입력 항목

| 항목 | 예시 | 필수 |
|------|------|------|
| 성명 | Nguyen Van An | ✅ |
| 국적 | VN (베트남) | ✅ |
| 공종 | rebar (철근) | ✅ |
| 선호 언어 | vi (베트남어) | ✅ |
| 이름 이니셜 | NVA | 권장 |
| 휴대폰 끝 4자리 | 5678 | 권장 |
| PIPA 동의 일시 | 2026-05-13 09:00 | 권장 |

### 공종 코드 목록

| 코드 | 한국어 |
|------|--------|
| rebar | 철근 |
| formwork | 거푸집 |
| concrete | 콘크리트 |
| scaffold | 비계 |
| welder | 용접 |
| signalman | 신호수 |
| equipment | 장비 |
| general | 일반 |
| foreman | 반장 |
| safety | 안전 |

### 예시 입력 (실제 현장 데이터 형식)

```
성명:          Nguyen Van An
국적:          VN
공종:          rebar
선호언어:      vi
이름이니셜:    NVA
휴대폰끝4자리: 5678
현장ID:        [관리자 계정에 자동 설정됨]
PIPA동의:      2026-05-13 09:00
```

→ **등록 완료 시 자동 부여:** `WRK-260513-0001` (근로자코드)

---

## Step 2. NFC 스티커 발급

**등록 완료 후 바로 이어서 진행**

1. 화면 하단 **"NFC 스티커 발급"** 버튼 클릭
2. Android 스마트폰을 NFC 스티커에 **밀착** (뒷면 중앙)
3. "기록 완료" 메시지 확인
4. 스티커를 근로자 안전모 또는 신분증 케이스에 부착

> NFC 기록 중 **핸드폰을 떼지 마세요.** 기록 실패 시 재발급 가능합니다.

### 발급 후 생성 정보

```
근로자코드:   WRK-260513-0001
NFC URL:      https://safe-link-v2.vercel.app/n/[workerId]?v=1&t=[issued_epoch]&s=[hmac_signature]
NDEF 크기:    138바이트 이하 (NTAG213 호환)
```

---

## Step 3. TBM NFC 세션 운영

**경로:** `관리자 → NFC 허브` (`/admin/nfc`)

### 3-1. TBM 세션 개설

1. **"새 TBM 세션 시작"** 클릭
2. 현장 선택 (자동 설정)
3. TBM 공지 선택 (선택사항)
4. 세션 제목 입력: `2026-05-13 오전 TBM`
5. 시작 → 상태: `running`

### 3-2. 근로자 NFC 태그 (참석 인증)

관리자 스마트폰 화면에서 **"NFC 스캔 대기"** 상태 유지 후,  
근로자가 자신의 **NFC 스티커를 관리자 폰에 태그.**

| 결과 메시지 | 의미 |
|-------------|------|
| ✅ checked_in | 정상 참석 기록 |
| ✅ already_certified | 이미 기록됨 (중복 태그, 무시) |
| ❌ sticker_revoked | 폐기된 스티커 → 재발급 필요 |

### 3-3. 세션 종료

모든 근로자 태그 완료 후 **"세션 종료"** 클릭 → 상태: `closed`

---

## Step 4. 근로자 재발급 (분실/훼손)

**경로:** `관리자 → 근로자 목록 → 해당 근로자 → 재발급`

1. 근로자 목록에서 이름/코드 검색
2. **"재발급"** 클릭 (이전 스티커 자동 폐기됨)
3. 새 NFC 스티커에 기록
4. 기존 스티커 폐기 처리 (회수 후 파기)

---

## 주의사항

- 스티커 1장 = 근로자 1명. **공유 사용 금지**
- 근로자가 퇴사하면 반드시 **비활성화** 처리 (목록 → 해당 근로자 → 비활성화)
- PIPA(개인정보보호법) 준수: 근로자 동의서 징구 후 등록

---

## 자주 묻는 질문

**Q. iOS(아이폰) 근로자는 NFC 태그가 안 되나요?**  
A. 아이폰은 NFC 읽기를 지원합니다. 근로자가 자신의 아이폰으로 스티커를 태그하면 Safari가 열리며 자동으로 언어 선택 → 입장합니다.

**Q. NFC가 안 되는 구형 폰은요?**  
A. 스티커 발급 시 QR코드도 동시 생성됩니다. QR 코드 출력 후 현장 게시판에 부착하면 카메라로 스캔 가능합니다.

**Q. 신규 근로자가 갑자기 투입됐을 때는?**  
A. 그 자리에서 바로 등록 → 스티커 발급 진행. 5분 이내 완료 가능합니다.
