# 안전장비 인센티브 지급 가이드
> 관리자(현장소장·안전관리자) 대상 / 청구항 12 기능

---

## 개요

TBM 안전 퀴즈에서 **80점 이상** 득점한 근로자에게 안전장비를 인센티브로 지급하고,  
지급 이력을 시스템에 자동 기록하는 기능입니다.

```
TBM 실시 → 퀴즈 발송 → 근로자 응답 → 80점↑ 근로자 조회 → 장비 지급 확정 → 이력 자동 저장
```

**관리자 접속 경로:** `SAFE-LINK 앱 → 관리자 메뉴 → 안전장비 인센티브`

---

## 사전 준비

| 항목 | 내용 |
|------|------|
| 계정 권한 | 관리자(admin) 이상 |
| 선행 작업 | TBM 세션 실시 + 퀴즈 발송 완료 |
| 준비물 | 지급할 안전장비 실물 |

> **퀴즈 발송 방법:** `관리자 메뉴 → 퀴즈 관리 → 해당 TBM 세션 선택 → 퀴즈 발송`

---

## Step 1. 인센티브 메뉴 진입

`관리자 → 안전장비 인센티브` (`/admin/incentive`)

화면에 **퀴즈 세션 목록**이 표시됩니다.

### 퀴즈 세션 목록 예시

```
┌─────────────────────────────────────────┐
│  퀴즈 세션                              │
│  ab3f1e2c…   [sent]                    │
│  2026-05-13 09:15                       │
├─────────────────────────────────────────┤
│  퀴즈 세션                              │
│  7c8d9a0b…   [sent]                    │
│  2026-05-12 08:50                       │
└─────────────────────────────────────────┘
```

→ **당일 세션** 클릭

---

## Step 2. 점수 확인 및 대상자 선정

세션 클릭 시 근로자별 점수가 **점수 내림차순**으로 표시됩니다.

### 점수 색상 의미

| 색상 | 점수 범위 | 인센티브 |
|------|-----------|----------|
| 🟢 녹색 | 80점 이상 | **지급 가능** |
| 🟡 황색 | 60~79점 | 지급 불가 |
| 🔴 적색 | 60점 미만 | 지급 불가 |

### 예시 화면 (5명 응답)

```
Nguyen Van An   WRK-260513-0001    95%  🟢  [장비 지급]
Sugrarov Ali    WRK-260513-0002    88%  🟢  [장비 지급]
Thida Myint     WRK-260513-0003    75%  🟡  점수 미달
Kim Dae Ho      WRK-260513-0004    65%  🟡  점수 미달
Sugruar Rahman  WRK-260513-0005        미응답
```

---

## Step 3. 장비 지급 확정

1. 지급 대상 근로자의 **[장비 지급]** 버튼 클릭
2. 팝업에서 **장비 유형 선택**

### 장비 유형 선택 예시 팝업

```
┌──────────────────────────────────┐
│  장비 유형 선택                  │
│                                  │
│  [안전모]  [안전화]  [안전조끼]  │
│  [안전장갑][안전안경][방진마스크] │
│  [방음귀마개][안전벨트][안전고리] │
│                                  │
│  [취소]         [지급 확정]      │
└──────────────────────────────────┘
```

3. 장비 선택 후 **[지급 확정]** 클릭
4. 버튼이 **✅ 지급완료** 로 변경되고 이력 저장 완료

---

## Step 4. 실물 장비 지급

시스템 기록 완료 후 **근로자에게 해당 장비 실물을 직접 지급**합니다.

| 근로자 | 코드 | 점수 | 지급 장비 | 지급 시각 |
|--------|------|------|-----------|-----------|
| Nguyen Van An | WRK-260513-0001 | 95% | 안전장갑 | 2026-05-13 10:05 |
| Sugrarov Ali | WRK-260513-0002 | 88% | 방진마스크 | 2026-05-13 10:07 |

> 위 표는 **예시입니다.** 실제 지급 이력은 시스템 하단 "지급 이력" 섹션에서 확인 가능합니다.

---

## 지급 이력 확인

세션 화면 하단 **"지급 이력"** 섹션에 자동 기록됩니다.

```
지급 이력
─────────────────────────────────
안전장갑     2026-05-13 10:05
방진마스크   2026-05-13 10:07
─────────────────────────────────
```

→ ESG 안전리포트의 **"안전장비 지급"** 항목에도 집계됩니다.

---

## 주의사항

- 동일 세션에서 같은 근로자에게 **중복 지급 불가** (시스템이 자동 차단)
- 지급 취소 기능 없음 — 잘못 지급 시 안전관리자에게 보고 후 수기 기록
- **80점 기준은 고정값**입니다 (코드 수정 없이는 변경 불가)

## 자주 묻는 질문

**Q. 근로자가 퀴즈에 응답을 안 했어요.**  
A. "미응답" 상태로 표시되며 인센티브 지급 대상에서 자동 제외됩니다. 재발송은 퀴즈 관리 메뉴에서 가능합니다.

**Q. 퀴즈 세션 목록이 비어 있어요.**  
A. TBM 세션 실시 후 퀴즈 발송 단계를 먼저 완료해야 합니다. `관리자 → 퀴즈 관리`에서 확인하세요.

**Q. 같은 근로자에게 다른 장비를 추가로 지급하고 싶어요.**  
A. 현재 세션당 근로자 1회 지급이 원칙입니다. 별도 세션에서 재지급 가능합니다.
