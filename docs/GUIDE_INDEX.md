## 업무별 빠른 찾기

**새 근로자가 오늘 투입됐다**  
→ [NFC 근로자 관리 > Step 1. 신규 등록](./GUIDE_NFC근로자관리.md)

**NFC 스티커를 잃어버렸다**  
→ [NFC 근로자 관리 > Step 4. 재발급](./GUIDE_NFC근로자관리.md)

**TBM 퀴즈 후 장비를 지급하고 싶다**  
→ [안전인센티브 > Step 2~3](./GUIDE_안전인센티브.md)

**발주처에 이번 달 안전 실적을 보고해야 한다**  
→ [ESG 안전리포트 > Step 4. JSON 내보내기](./GUIDE_ESG안전리포트.md)

**ESG 점수가 낮게 나왔다**  
→ [ESG 안전리포트 > ESG 점수 계산 방식](./GUIDE_ESG안전리포트.md)

## 관리자 접속 경로 요약

```
SAFE-LINK 앱 로그인 (관리자 계정)
│
├─ 관리자 메뉴
│   ├─ NFC 허브              → NFC 근로자 관리 & TBM 세션
│   ├─ 근로자 목록/등록      → 신규 등록, 재발급, 비활성화
│   ├─ 안전장비 인센티브     → 퀴즈 결과별 장비 지급
│   └─ ESG 안전 리포트       → 기간별 안전 지표 집계
```
