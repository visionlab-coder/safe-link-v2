## ① OpenAI API (Whisper 음성인식)

### 과금 구조 (종량제)
- 단가: **$0.006 / 분** (약 8원/분)
- 사용한 분량만큼만 과금, 기본료·월정액 없음
- 신규 가입 시 **$5 무료 크레딧 자동 지급** (3개월 유효)

### 실사용 예상
| 사용량 | 월 과금액 |
|---|---|
| 테스트(1시간 미만) | **약 500원** (무료 크레딧으로 충당 가능) |
| 파일럿(10시간) | 약 5,000원 |
| 전체 운영(30개 현장, 하루 10분) | **약 3~6만원** |

### 가입 절차

**1단계 — 회원 가입**
1. https://platform.openai.com/signup 접속
2. 이메일: `visionlab@seowonenc.co.kr` (회사 계정)
3. 휴대폰 인증 (김무빈 차장 본인)

**2단계 — 결제 수단 등록 (결재 직원 담당)**
1. 로그인 후 좌측 메뉴 → **Settings → Billing**
2. **Add payment method** 클릭
3. 법인 신용카드 정보 입력
   - 카드번호 / 유효기간 / CVC
   - 청구 주소: 서원토건 본사 주소 (영문)
   - Tax ID: 법인 사업자번호 (선택사항, 부가세 면제 혜택 위해 입력 권장)
4. 카드 인증 위해 **$5 임시 결제 후 즉시 환불** (정상 절차)

**3단계 — 비용 한도 설정 (필수, 폭주 방지)**
1. Settings → Billing → **Usage limits**
2. **Monthly budget limit: $50** (약 6.5만원)
3. **Email alert: $25** (50% 도달 시 김무빈 차장 메일 통지)
4. 한도 초과 시 API 자동 차단 → 추가 과금 불가

**4단계 — API 키 발급 (김무빈 차장 담당)**
1. 좌측 메뉴 → **API keys → Create new secret key**
2. Name: `safe-link-v2-prod`
3. `sk-proj-...` 형태 키 생성 → 복사 (창 닫으면 재조회 불가)
4. Vercel 환경변수에 등록

### 비고
- 결제 통화: **USD (달러)** — 매월 환율 적용 후 청구
- 청구서: 월초 이메일 자동 발송
- 영수증: Billing → Invoices에서 PDF 다운로드 가능

## ② Naver Papago API (번역)

### 과금 구조
- **월 1천만 글자 무료** (건설 TBM 기준 사실상 무제한)
- 초과 시: 1글자당 0.02원 (100만자 = 2만원)
- **실질 과금 0원 예상**

### 가입 절차

**1단계 — 네이버클라우드 회원 가입**
1. https://www.ncloud.com 접속
2. 우측 상단 **회원가입** → **사업자 회원**
3. 정보 입력:
   - 사업자등록번호: 서원토건 법인번호
   - 대표자명: 김진현
   - 담당자: 김무빈 차장 (visionlab@seowonenc.co.kr)
4. 사업자등록증 사본 업로드
5. 승인 대기 (영업일 기준 1일 이내)

**2단계 — 결제 수단 등록 (결재 직원 담당)**
1. 콘솔 로그인 → 우측 상단 프로필 → **마이페이지**
2. **결제수단 관리 → 카드 등록**
3. 법인 신용카드 등록 (국내카드 가능)
4. 자동결제 동의

**3단계 — Papago Translation 서비스 신청**
1. 콘솔 상단 검색창에 **Papago Translation** 입력
2. **이용 신청** → 약관 동의 → 서비스 활성화 (즉시)

**4단계 — Application 등록 (김무빈 차장 담당)**
1. Papago Translation → **Application 등록**
2. 정보 입력:
   - Application 이름: `safe-link-v2`
   - 서비스 환경: **WEB**
   - 서비스 URL: `https://safe-link-v2.vercel.app`
3. 등록 완료 후 앱 상세 페이지에서 확인:
   - **Client ID** (공개 키)
   - **Client Secret** (비공개 키)
4. Vercel 환경변수에 등록

### 비고
- 결제 통화: **KRW (원화)** — 세금계산서 자동 발행
- 사업자 회원은 세금계산서 수령 가능 (월말 자동 발행)
