# ESG 안전 리포트 사용 가이드
> 관리자(현장소장·안전관리자·공무) 대상 / 청구항 24 기능

---

## 개요

현장의 안전 활동 데이터를 **ESG(환경·사회·지배구조) 지표**로 자동 집계하는 리포트입니다.  
TBM 참석, 안전서약, 작업중지, 장비 지급, 통역 세션 등을 한눈에 파악할 수 있으며  
발주처·본사 보고용 JSON 파일로 내보내기(Export)가 가능합니다.

**관리자 접속 경로:** `SAFE-LINK 앱 → 관리자 메뉴 → ESG 안전 리포트`

---

## 사전 준비

| 항목 | 내용 |
|------|------|
| 계정 권한 | 관리자(admin) 이상 |
| 데이터 조건 | 해당 기간 내 TBM 세션 1건 이상 실시되어야 의미 있는 수치 생성 |
| 권장 주기 | 월 1회 (매월 1일 전월 리포트 생성) |

---

## Step 1. 현장 및 기간 선택

`관리자 → ESG 안전 리포트` (`/admin/esg`)

### 선택 화면

```
┌──────────────────────────────────────────┐
│  현장 선택                               │
│  [▼ SW-001 서원 인천계양 현장]           │
│                                          │
│  From         To                         │
│  [2026-04-01] [2026-04-30]              │
│                                          │
│  [🔄 리포트 생성]   [📥 JSON]           │
└──────────────────────────────────────────┘
```

### 현장 선택 안내

| 권한 | 표시 현장 |
|------|-----------|
| HQ_ADMIN / SUPER_ADMIN | 전체 현장 선택 가능 |
| 일반 admin | 본인 담당 현장만 표시 |

### 권장 기간 설정 예시

| 목적 | From | To |
|------|------|----|
| 전월 월간 리포트 | 2026-04-01 | 2026-04-30 |
| 당월 중간 점검 | 2026-05-01 | 2026-05-13 |
| 분기 리포트 | 2026-04-01 | 2026-06-30 |

---

## Step 2. 리포트 생성

**[🔄 리포트 생성]** 클릭 → 약 2~3초 후 결과 표시

---

## Step 3. 결과 해석

### 실제 현장 예시 리포트 (2026년 4월, 인천계양 현장)

```
기간: 2026-04-01 ~ 2026-04-30

┌─────────────────┐ ┌──────────────────┐
│ TBM 세션        │ │ 이수 인증율      │
│      22         │ │      87%         │
│ 참석 346명      │ │ 302/346명   🟢   │
│ ████████████ 87%│ │ ████████████ 87% │
└─────────────────┘ └──────────────────┘

┌─────────────────┐ ┌──────────────────┐
│ 안전 서약       │ │ 작업중지 요청    │
│      89         │ │       3          │
│ 서명완료 82건   │ │ 해결 3/3    🟢   │
│ ████████████ 92%│ │ ████████████100% │
└─────────────────┘ └──────────────────┘

┌─────────────────┐ ┌──────────────────┐
│ 감사 체인 기록  │ │ 실시간 통역      │
│     1,247       │ │       18         │
│ SHA-256 해시    │ │ 다국어 세션      │
└─────────────────┘ └──────────────────┘

안전장비 지급: 17건
─────────────────────────────────────────
ESG 종합 점수

        91 / 100

TBM인증률(40) + 서약서명률(30) + 작업중지해결률(20) + 세션실시(10)
```

### ESG 점수 계산 방식

| 항목 | 배점 | 계산 방법 |
|------|------|-----------|
| TBM 이수 인증율 | 40점 | 인증 완료자 ÷ 참석자 × 40 |
| 안전서약 서명율 | 30점 | 서명 완료 ÷ 전체 서약 × 30 |
| 작업중지 해결율 | 20점 | 해결건 ÷ 총 요청건 × 20 |
| TBM 세션 실시 여부 | 10점 | 1건 이상 실시 시 10점 |

### 점수 판정 기준

| 점수 | 등급 | 의미 |
|------|------|------|
| 90~100 | 🟢 우수 | 발주처 제출 가능 |
| 70~89 | 🟡 보통 | 개선 필요 항목 확인 |
| 70 미만 | 🔴 미흡 | TBM 이수율·서약 관리 점검 필요 |

## Step 4. JSON 내보내기 (발주처·본사 보고)

**[📥 JSON]** 버튼 클릭 → 파일 자동 다운로드

### 다운로드 파일명 형식

```
esg_report_[현장ID]_2026-04-01_2026-04-30.json
```

  },

| 일자 | 담당 | 작업 |
|------|------|------|
| 매월 1일 | 공무/안전관리자 | 전월 리포트 생성 → JSON 저장 |

---

## 주의사항

- 리포트는 **현재 DB 데이터를 실시간 집계**합니다. 과거 소급 수정 불가.
- TBM 세션이 없는 기간은 TBM 항목이 0으로 표시됩니다 (휴일 등).
- 작업중지 요청이 0건이면 해결율을 **100%로 계산** (분모 0 예외 처리).

## 자주 묻는 질문

**Q. 리포트가 비어 있어요.**  
A. 현장과 기간을 선택 후 **[리포트 생성]** 버튼을 눌러야 집계됩니다.

**Q. 감사 체인 기록이 많을수록 좋은 건가요?**  
A. 네. 안전서약·작업중지 등 중요 이벤트마다 SHA-256 해시가 체인으로 누적됩니다. 숫자가 많을수록 활동량이 많은 것입니다.

**Q. ESG 점수를 높이려면 어떻게 해야 하나요?**  
A. 가중치 순: ① TBM 이수 인증율 관리(40점) → ② 안전서약 서명 독려(30점) → ③ 작업중지 즉시 해결(20점) → ④ TBM 세션 빠짐없이 실시(10점)
