# SAFE-LINK 실생활 현장 번역 지침서 (V1.0)

## 1. 번역의 대원칙
사전적, 기계적 번역보다 **현장 실무에서의 실질적 의미 전달**을 최우선으로 한다. 외국인 근로자가 한국 관리자에게, 혹은 관리자가 근로자에게 말할 때 이질감이 없어야 하며, 존중의 의미(존댓말)가 반드시 포함되어야 한다.

## 2. 주요 문구 매핑 가이드 (Korean Standard)

| 원문(Mechanical) | 실생활 번역(Natural/Contextual) | 비고 |
| :--- | :--- | :--- |
| **묻다 / 질문하다** | **물어보고 싶은 게 있습니다.** | "请问一下" 등 질문 시작 시 사용 |
| **말하다 / 보고하다** | **말씀드릴 게 있습니다.** | 보고 또는 대화 시작 시 |
| **도와줘 / 돕다** | **도움이 필요합니다.** | 지원 요청 시 |
| **위험** | **위험하니 주의하시기 바랍니다.** | 안전 경고 시 |
| **조심해 / 주시해** | **조심하시기 바랍니다.** | 작업 중 주의 환기 |
| **기다려** | **잠시만 기다려 주세요.** | 대기 요청 시 |
| **안녕 / 반가워** | **안녕하세요.** | 인사 |
| **응 / 그래 / 오케이** | **네 / 알겠습니다.** | 긍정 답변 |
| **아니 / 싫어** | **아니요 / 아닙니다.** | 부정 답변 |

## 3. 언어별 적용 지침 (Global Application)
이 지침은 중국어(ZH)뿐만 아니라 베트남어(VI), 태국어(TH), 우즈베크어(UZ) 등 **시스템이 지원하는 모든 16개 언어**에 공통 적용된다. 
- 기계 번역 결과가 동사 원형(Root Verb, 예: '묻다')으로 나올 경우, 시스템의 `formalizeKo` 유틸리티를 거쳐 반드시 실생활 문장으로 변환한다.

## 4. 음성 출력(TTS) 지침
- **성별 일치**: 사용자가 선택한 성별(Male/Female)에 맞는 Voice Data를 브라우저에서 최우선 추출한다.
- **자연스러운 톤**: 단순 피치 조절이 아닌, 각 언어별 'Natural' 보이스를 우선 사용하여 기계음을 최소화한다.
- **현장 가청도**: 현장 소음 속에서도 잘 들릴 수 있도록 속도는 0.95배, 볼륨은 1.0(최대)으로 유지한다.

*본 지침은 시스템의 `src/utils/politeness.ts` 및 `src/utils/tts.ts` 코드에 로직으로 반영되어 강제 적용됨.*
