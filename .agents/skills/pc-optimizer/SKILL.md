# PC Optimizer Skill

Windows 환경의 리소스를 분석하고 최적화하기 위한 스킬입니다.

## 기능
1. **진단 (Diagnose):** 메모리, CPU 점유율 분석 및 응답 없는 프로세스 확인.
2. **최적화 (Optimize):** 불필요한 프로세스 종료 및 임시 파일 정리.

## 사용 가이드라인
- **안전 우선:** 시스템 필수 프로세스는 종료하지 않습니다.
- **사용자 승인:** 모든 종료 및 삭제 작업은 실행 전 사용자에게 보고하고 승인을 받아야 합니다.
- **Surgical Cleanup:** 필요한 앱만 타겟팅하여 정리합니다.

## 스크립트 위치
- `scripts/diagnose.ps1`: 시스템 상태 점검
- `scripts/cleanup.ps1`: 리소스 정리
